## 🧠 Part 1 — Distributive Law

(b + c) × a = b × a + c × a

text

இதுதான் நாம் 42 × 7 செய்யும்போது பயன்படுத்தியது.
42 = 40 + 2

(40 + 2) × 7 = 40 × 7 + 2 × 7

text

👉 இது distributive law.

# சரி 👍

இப்போது நீங்கள் கேட்டது:

**(z+d)²** expand செய்யும் போது  
எப்படி **z² + 2zd + d²** வந்தது?

இதை மிக மெதுவாக, step-by-step பார்க்கலாம்.

----------

### 🧠 Step 1 — Square என்றால் என்ன?

**(z+d)²**

இதன் அர்த்தம்:

**(z+d)(z+d)**

அதாவது:

**(z+d) × (z+d)**

----------

### 🧠 Step 2 — Distributive Law apply

Rule:

**(a+b)c = ac + bc**

இப்போது:

**(z+d)(z+d)**

இதை:

👉 முதல் bracket (z+d)  
👉 இரண்டாவது bracket (z+d)

முதல் bracket உள்ள ஒவ்வொன்றையும்  
இரண்டாவது bracket முழுவதுடனும் multiply செய்கிறோம்.

First distribute

**(z+d)(z+d)**

= **z(z+d) + d(z+d)**

(முதல் bracket உள்ள z, d இரண்டும் distribute ஆனது)

----------

### 🧠 Step 3 — மீண்டும் distribute

இப்போது:

**z(z+d)**

= z² + zd

அதேபோல்:

**d(z+d)**

= dz + d²

----------

### 🧠 Step 4 — எல்லாம் சேர்த்து எழுதுங்கள்

z² + zd + dz + d²

ஆனால்:

**zd = dz**

அதனால்:

z² + zd + zd + d²

----------

### 🧠 Step 5 — combine like terms

zd + zd = 2zd

**Final:**

**z² + 2zd + d²**

----------

### 📐 Visual box method (FOIL idea)

text

 -------------
z   |  z²   |  zd  |
d   |  dz   |  d²  |

Add all four:

z² + zd + dz + d²

= **z² + 2zd + d²**

----------

### 🔎 Real example

**41²**

= (40+1)²

z = 40  
d = 1

Formula:

z² + 2zd + d²

= 1600 + 80 + 1  
= **1681**

----------

### 🧠 Why 2zd வந்தது?

ஏனெனில்:

👉 z×d ஒரு முறை

இரண்டும் same value

அதனால் **2 × zd**
