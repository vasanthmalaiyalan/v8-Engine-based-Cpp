# 🔹 Subtraction Method — Core Idea

ஒரு number 8 அல்லது 9ல் முடிந்தால்:

59 → 60 − 1  
88 → 90 − 2  
96 → 100 − 4

அதாவது:

ab = (base − small number)

பிறகு:

base × other number  
− (small number × other number)

----------

# 📌 Example 1

59 × 17

59 = 60 − 1

= 60×17 − 1×17  
= 1020 − 17  
= **1003**

----------

# 📌 Example 2

89 × 72

89 = 90 − 1

= 90×72 − 1×72  
= 6480 − 72  
= **6408**

----------

# 📌 Example 3

88 × 23

88 = 90 − 2

= 90×23 − 2×23  
= 2070 − 46  
= **2024**

----------

# 📌 Example 4 (round to 100)

96 × 73

96 = 100 − 4

= 100×73 − 4×73  
= 7300 − 292  
= **7008**

----------

# 📌 Example 5

88 × 76

88 = 90 − 2

= 90×76 − 2×76  
= 6840 − 152

Now:

6840 − 152  
= 6640 + 48  
= **6688**

(அல்லது complement method)

----------

# 📌 Example 6

67 × 59

= 60×67 − 1×67  
= 4020 − 67  
= **3953**

# 🔹 Final Summary

Subtraction Method useful when:

-   number ends in 8 or 9
    
-   number near 100
    
-   rounding makes multiplication easier
    

Formula style:

ab × cd  
= base×cd − small×cd
