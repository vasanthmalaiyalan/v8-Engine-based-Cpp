## Part 1 — Distributive Law (Multiplication Tricks)

**Book சொல்கிறது:**
```
(b + c) × a = (b × a) + (c × a)
```

**தமிழில்:**
```
கூட்டத்தை × செய்தால்
ஒவ்வொன்றையும் × செய்து கூட்டலாம்
```

---

### 🔎 Example — 42 × 7

```
42 = 40 + 2

(40 + 2) × 7 = 40×7 + 2×7
             = 280 + 14
             = 294
```

---

### ⭐ Why True (Intuitive)

```
7 bags ஒவ்வொன்றிலும் 42 coins

42 = 40 gold + 2 silver

gold total  = 40 × 7
silver total = 2 × 7
total        = (40×7) + (2×7)
```

✔ அதனால் law correct

---

### 🧠 3-Digit Case — 326 × 7

```
326 = 300 + 20 + 6

(300 + 20 + 6) × 7
= 300×7 + 20×7 + 6×7
= 2100 + 140 + 42
= 2282
```

✔ இதுதான் 3-by-1 trick

# 📐 Terms Transpose — Visual Balance Model

## 🧠 Core Idea

```
Left Side  =  Right Side
```

## ⚖️ Visual Balance Model

```
          ⚖️
  (A+d)(A-d)   =   A² - d²
```

✔ இருபுறமும் சமம்

---

## 🧠 Goal

**A² மட்டும் ஒரு பக்கம் இருக்க வேண்டும்.**

**இப்போது RHS-ல்:**
```
A² - d²
```

✔ அதாவது A²-க்கு பக்கத்தில் −d² உள்ளது

---

## ⭐ Rule — Transpose என்றால் என்ன?

```
"Transpose" என்று சொல்வது உண்மையில்:

❌ ஒரு term-ஐ மறுபக்கம் அனுப்புவது அல்ல
✔ இருபுறமும் அதன் inverse operation செய்வது
```

---

## 📐 Step — +d² இருபுறமும் சேர்க்க

```
(A+d)(A-d) + d²  =  A² - d² + d²
```

---

## 🧠 RHS Simplify

```
-d² + d² = 0
```

**அதனால்:**
```
(A+d)(A-d) + d²  =  A²
```

✔ இப்போது A² isolate ஆகிவிட்டது
